# Instalog SDK Documentation

## Installation
Add the Instalog dependency to your project.

### Gradle (build.gradle)
```kotlin
dependencies {
    implementation("dev.instalog:mobile:1.0.1")
}
```

### Maven (pom.xml)
```xml
<dependency>
    <groupId>dev.instalog</groupId>
    <artifactId>mobile</artifactId>
    <version>1.0.1</version>
</dependency>
```
